🚽 Cagadas App: El Ranking Definitivo de Deposiciones

¡Bienvenido a Cagadas App! La aplicación social y de gamificación que convierte una necesidad biológica en una competencia épica. Rastrea, puntúa y clasifica las "cagadas" de tu equipo o grupo de amigos. ¿Quién se alzará con el título del "Maestro Defecador"?

🌟 Características Principales

Registro de Eventos: Sistema sencillo para registrar cada "evento" con fecha y hora.

Ranking en Tiempo Real: Visualiza quién lidera la tabla de posiciones con más eventos registrados.

Sistema de Calificación (Gamificación): Puntuación basada en factores de calidad y rareza (ver detalles abajo).

Sistema de Recompensas: Desbloquea títulos y logros virtuales para incentivar la participación.

🏆 Sistema de Calificación y Puntuación

La puntuación no se trata solo de la cantidad, sino también de la calidad y la épica de la hazaña. Cada registro recibe una puntuación base de +10 puntos y se pueden añadir bonificaciones (o penalizaciones) con un sistema de tags:

🛠️ Instalación del Proyecto

Requisitos Previos

Necesitas tener instalado:

Node.js y npm: Asegúrate de tener una versión reciente (v18 o superior).

Git: Para clonar el repositorio.

Pasos de Instalación

Sigue estos sencillos pasos para tener la Cagadas App funcionando en tu entorno local.

Clonar el Repositorio de GitHub:
Abre tu terminal o línea de comandos y ejecuta el siguiente comando:

git clone [https://github.com/lucas-sanchez-cabrera/Proyecto-Cagadas.git)


Luego, navega al directorio del proyecto:

cd ProyectoCagadas


Instalar Dependencias de Node.js:
La aplicación utiliza npm (incluido con Node.js) para gestionar las librerías necesarias.

npm install


Configuración de Variables de Entorno (Opcional, pero recomendado):
Crea un archivo llamado .env en la raíz del proyecto y añade las variables de configuración necesarias (por ejemplo, claves de bases de datos, si aplica).

Ejecutar la Aplicación:
Inicia el servidor de desarrollo. Este comando variará dependiendo de cómo esté configurado tu archivo package.json (comúnmente start o dev).

Debes moverte a estas ramas y ejecutar las 2:

cd Proyecto-Cagadas/Cagadas-FrontEnd
cd Proyecto-Cagadas/Cagadas-BackEnd

npm run start
# O, si tienes un script de desarrollo:
# npm run dev


La aplicación estará accesible en tu navegador, probablemente en http://localhost:3000 (el puerto puede variar).
